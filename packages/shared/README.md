# @ktjs/shared

Internal shared utilities and types for kt.js monorepo.

**This package is private and not published to npm.**

## Contents

- `src/lib/` - Shared utility functions, constants, and DOM helpers
- `src/types/` - Shared TypeScript type definitions

## Usage

This package is only used internally within the kt.js monorepo. Code from this package will be bundled into the published packages (@ktjs/core, @ktjs/router, etc.).
